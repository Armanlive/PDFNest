## ğŸš€ Getting Started

### Prerequisites

- **Node.js** (version 16 or higher)
- **npm** or **yarn** package manager
- Modern web browser

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/pdfnest.git
   cd pdfnest
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
